# KashiGlitz: Consumer Loyalty Analysis for Nykaa (Varanasi)

### 📊 Project Overview
An interactive data analytics project exploring consumer behavior in **Varanasi (Kashi)**. Using the **S-O-R (Stimulus-Organism-Response) Model**, this study quantifies how Product Quality, Perceived Value, and Advertisements influence brand loyalty.

### 🛠️ Tech Stack
* **Tool:** Microsoft Excel (Advanced)
* **Features:** Pivot Tables, Interactive Slicers, Radar Charts, KPI Dashboards.
* **Sample Size:** 105 Respondents (Varanasi Market).

### 📈 Key Insights
* **Top Driver:** Product Quality & Ads Appeal (Correlation: 0.75).
* **Demographic:** High engagement in the 21-30 age group.
* **Outcome:** Strategic recommendations for Tier-2 market penetration.
